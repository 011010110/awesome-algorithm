###7. Reverse Integer

题目:
<https://leetcode.com/problems/Reverse-Integer/>


难度:

Easy


思路

如果输入的是负数，就递归调用原函数，参数变成-x即可
最后要处理overflow


```python
class Solution(object):
    def reverse(self, x):
        """
        :type x: int
        :rtype: int
        """
        if x < 0:
            return -self.reverse(-x)
        res = 0
        while x:
            res = res * 10 + x % 10
            x /= 10
        return res if res <= 0x7fffffff else 0
```

```python
class Solution(object):
    def reverse2(self, x):
        """
        :type x: int
        :rtype: int
        """
        if x < 0:
            x = int(str(x)[::-1][-1] + str(x)[::-1][:-1])
        else:
            x = int(str(x)[::-1])
        x = 0 if abs(x) > 0x7FFFFFFF else x
        return x
```

class Solution(object):

        :type x: int
        :rtype: int
