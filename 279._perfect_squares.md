
### 279. Perfect Squares


题目:
<https://leetcode.com/problems/perfect-squares/>


难度:

Medium


DP, 状态转移方程:

```dp[i] = min(dp[i], dp[i - j * j] + 1)```




```python
class Solution(object):
    def numSquares(self, n):
        """
        :type n: int
        :rtype: int
        """
        dp = [0] * (n+1)
        for i in range(n+1):
            dp[i] = i
            j = 1
            while j * j <= i:
                dp[i] = min(dp[i], dp[i-j*j] + 1)
                j += 1
        return dp[-1]

```
